# Company Landing Page

This project is a simple company landing page created as part of **Experiment-2**.  
It demonstrates the use of semantic HTML and CSS to build a clean, structured, and visually appealing webpage.

## Project Structure

company_landing_page/
├── index.html
├── style.css
└── screenshots/

## Features

- Hero section with company name and tagline  
- About section describing the company  
- Services section with multiple service cards  
- Testimonials section with customer feedback  
- Contact section with company details and a simple contact form  
- Clean and readable UI design  

## Technologies Used

- HTML5  
- CSS3
